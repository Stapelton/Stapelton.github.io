Política de Privacidad

Última actualización: Enero 2026

1. Información que Recopilamos

Aviso Balizas V16 recopila y utiliza la siguiente información:

  • Ubicación GPS: La aplicación utiliza la ubicación de su dispositivo para detectar cuando se acerca a una baliza V16 y proporcionarle alertas de seguridad. También utiliza la velocidad del vehículo para optimizar el rendimiento.
  • Datos de Balizas: La aplicación descarga información sobre balizas V16 activas desde la API oficial de DGT 3.0 para mostrarle alertas en tiempo real.
  • Datos de Incidencias: Información sobre obras, accidentes, atascos y otras incidencias desde la API oficial de DGT 3.0.
  • Datos de Electrolineras: Información sobre puntos de carga para vehículos eléctricos desde fuentes públicas con licencia Creative Commons Attribution.
  • Datos de Gasolineras: Información sobre estaciones de servicio desde la API oficial del Ministerio para la Transición Ecológica y el Reto Demográfico (Gobierno de España).
  • No recopilamos información personal identificable, datos de contacto, ni información de terceros.

2. Uso de la Información

La información recopilada se utiliza exclusivamente para:

  • Proporcionar alertas de seguridad cuando se acerca a una baliza V16.
  • Mostrar la ubicación de balizas, incidencias, electrolineras y gasolineras en el mapa.
  • Calcular distancias para las notificaciones de proximidad.
  • Optimizar el rendimiento de la aplicación mediante gestión automática de recursos (desactivación automática de capas cuando el vehículo está en marcha).
  • Mejorar la funcionalidad de la aplicación.
No compartimos, vendemos ni alquilamos su información a terceros.

3. Permisos de la Aplicación

La aplicación requiere los siguientes permisos:

  • Ubicación (GPS): Necesario para detectar su posición y calcular distancias a las balizas V16. Este permiso es esencial para el funcionamiento de la aplicación.
  • Ubicación en segundo plano: Necesario para continuar monitoreando su ubicación incluso cuando la aplicación está cerrada, permitiendo alertas continuas.
  • Notificaciones: Necesario para mostrar alertas de voz y texto cuando se acerca a una baliza V16.
  • Internet: Necesario para descargar información actualizada de balizas V16.

4. Almacenamiento de Datos

  • Los datos de ubicación se procesan localmente en su dispositivo.
  • La información de balizas e incidencias se actualiza cada 2 minutos desde la API oficial.
  • Los datos de electrolineras se descargan bajo demanda según la zona visible en el mapa.
  • No almacenamos su historial de ubicaciones de forma permanente.
  • No enviamos datos de ubicación a servidores externos.
  • Todos los datos en caché se almacenan localmente en su dispositivo y no se comparten con terceros.

5. Seguridad

Implementamos medidas de seguridad para proteger su información:

  • Procesamiento local de datos de ubicación.
  • Comunicación segura con las APIs oficiales (DGT 3.0, Ministerio para la Transición Ecológica).
  • Almacenamiento local seguro de caché en su dispositivo.
  • No almacenamiento de datos personales sensibles.
  • Optimización de recursos para proteger la privacidad y el rendimiento del dispositivo.

6. Sus Derechos

Usted tiene derecho a:

  • Revocar los permisos de ubicación en cualquier momento desde la configuración de su dispositivo.
  • Desactivar las notificaciones desde la configuración de la aplicación.
  • Desinstalar la aplicación en cualquier momento.
  • Solicitar información sobre los datos que procesamos (aunque no almacenamos datos personales identificables).

7. Cambios en esta Política

Nos reservamos el derecho de actualizar esta política de privacidad. Le notificaremos de cualquier cambio publicando la nueva política en esta página y actualizando la fecha de "Última actualización".

8. Contacto

Si tiene preguntas sobre esta política de privacidad, puede contactarnos a través de la información de contacto disponible en Google Play Store.
